# Sales Dashboard using Power BI

## 📊 Project Overview
This project focuses on analyzing sales data using Power BI to understand customer behavior and business performance.

## 🛠️ Tools Used
- Power BI

## 📈 Key Features
- KPI metrics (Total Sales, Orders, Customers)
- Category-wise and region-wise analysis
- Monthly sales trends
- Interactive filtering using slicers

## 📌 Note
The dashboard was created using Power BI by combining multiple datasets (Customers, Orders, Products) and building relationships between them.
